How to Play A26

1. Six houses on the table: A, 2, 3, 4, 5, 6.

2. Select one or more houses and bet between ₹2,000 and ₹60,000 per house. You can bet on multiple houses in the same round — each house has its own independent ₹60,000 cap.

3. The dealer shuffles the deck, lets a player cut, then draws 3 random cards.

4. If a card's value matches the house you bet on, you win!

1 Match1 : 1
2 Matches1 : 2
3 Matches1 : 4

Example: Bet ₹2,000 on house "A". If 1 of the 3 cards is an Ace, you win ₹4,000 (1:1). If 2 are Aces, you win ₹6,000 (1:2). If all 3 are Aces, you win ₹8,000 (1:4).

Card Values: A = Ace · 2-10 = face value · J, Q, K count as their rank only when house matches.
